# 💰 Personal Expense Tracker

A web-based expense tracking app built with Python and Streamlit. Users can log expenses by category, view them in a live-updating table, and see spending breakdowns via bar charts.

## Features
- Add expenses with amount, category, date, and notes
- View all expenses in a searchable table
- Real-time total spending and category-wise bar chart

## Tech Stack
- Python
- Streamlit
- Pandas

## How to Run
\`\`\`bash
pip install streamlit pandas matplotlib
streamlit run app.py
\`\`\`
